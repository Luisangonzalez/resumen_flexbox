# Flex direction

Flex wrap apila los elementos en pila tipo block cuando no hay espacio, si queremos apilar los elementos a√∫n habiendo espacio debemos utlizar la propiedad `flex-direction` con el valor `column`.

```CSS
.setup {
display: flex;
flex-direction: column;
}
```
El valor por defecto es `row`, estos son los posibles valores:
* `row`
* `row-reverse`
* `column`
* `column-reverse`
